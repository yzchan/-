int 21h是DOS所提供的中断例程所提供的子程序如下
```
00H：程序终止(同INT 20H)
01H：键盘输入并回显
02H：显示输出
03H：异步通迅输入
04H：异步通迅输出
05H：打印机输出
06H：直接控制台I/O
07H：键盘输入(无回显)
08H：键盘输入(无回显) 检测Ctrl-Break
09H：显示字符串
0AH：键盘输入到缓冲区
0BH：检验键盘状态
0CH：清除输入缓冲区并 请求指定的输入功能
0DH：磁盘复位
0EH：指定当前缺省的磁盘驱动器
0FH：打开文件
10H：关闭文件
11H：查找第一个目录项
12H：查找下一个目录项
13H：删除文件
14H：顺序读
15H：顺序写
16H：建文件
17H：文件改名
19H：取当前缺省磁盘驱动器
1AH：置DTA地址
1BH：取缺省驱动器FAT信息
1CH：取任一驱动器FAT信息
21H：随机读
22H：随机写
23H：测定文件大小
24H：设置随机记录号
25H：设置中断向量
26H：建立程序段前缀
27H：随机分块读
28H：随机分块写
29H：分析文件名
2AH：取日期
2BH：设置日期
2CH：取时间
2DH：设置时间
2EH：置磁盘自动读写标志
2FH：取磁盘缓冲区的首址
30H：取DOS版本号
31H：结束并驻留
33H：Ctrl-Break检测
35H：取中断向量
36H：取空闲磁盘空间
38H：置/取国家信息
39H：建立子目录(MKDIR)
3AH：删除子目录（RMDIR）
3BH：改变当前目录(CHDIR)
3CH：建立文件
3DH：打开文件
3EH：关闭文件
3FH：读文件或设备
40H：写文件或设备
41H：删除文件
42H：移动文件指针
43H：置/取文件属性
44H：设备文件I/O控制
45H：复制文件代号
46H：人工复制文件代号
47H：取当前目录路径名
48H：分配内存空间
49H：释放内容空间
4AH：调整已分配的存储块
4BH：装配/执行程序
4CH：带返回码结束
4DH：取返回代码
4EH：查找第一个匹配文件
4FH：查找下一个匹配文件
54H：取盘自动读写标志
56H：文件改名
57H：置/取文件日期和时间
58H：取/置分配策略码
59H：取扩充错误码
5AH：建立临时文件
5BH：建立新文件
5CH：控制文件存取
62H：取程序段前缀
```
